# BiliRetro

> 一款针对安卓 5.0+ 设备的复古风格 B站第三方客户端

##  简介

BiliRetro 是一款轻量级的 B站第三方客户端，专为低版本安卓设备（Android 5.0+）优化。界面风格模仿 2014 年前后的 B站客户端，力求简洁流畅。

##  功能

- 首页分区视频浏览（热门、动画、音乐、游戏、科技、生活）
- 视频播放（支持 ExoPlayer）
- 评论查看与点赞
- 视频收藏与投币
- 搜索视频
- Token 登录与导入（跨设备登录）
- 横竖屏自适应

##  技术栈

- 语言：Java
- 播放器：ExoPlayer
- 网络：OkHttp
- 图片加载：Glide
- 二维码：ZXing
